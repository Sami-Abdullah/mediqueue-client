# MediQueue 🏥

A full-stack tutor booking platform built with Next.js, Express, and MongoDB. Students can browse tutors, book sessions, and manage their learning schedule.

## Tech Stack

**Frontend** — Next.js 15 (App Router), Tailwind CSS, DaisyUI, Better Auth  
**Backend** — Express.js, MongoDB, Better Auth  
**Deployment** — Vercel (both frontend and backend)

## Features

- Browse and search tutors by subject
- View detailed tutor profiles with availability, pricing, and experience
- Book sessions with preferred date, time, and session type
- View and manage your booked sessions
- Authentication (sign up, sign in, sign out) with session management
- Profile management with avatar support
- Dark / light theme toggle
- Slot availability tracking — prevents overbooking


### Prerequisites
- Node.js 18+
- MongoDB Atlas account
- Two Vercel projects (or run locally)
